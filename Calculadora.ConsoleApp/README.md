# Calculadora

## Projeto
Desenvolvido drante o curso Back-End da [Acaemia do Programador](https://www.academiadoprogramador.net) 2026 

## Introdução

Uma calculadora de console simples mas poderosa que permite realizar as quatro operações matemáticas, além de visualização da tabuada.

## Funcionalidades

- **Operações Básicas:** Realize somas, subrações, multiplicações e divisões com facilidade!
- **Tabuada:** A calculadora é capaz de gerar a tabuada de um númeroinformado até outro número informado (que será o mltiplicador).
- **Histórico de Operações:** A calculadora é capaz de armazenar na memória um histórico das operaões anteriores.

## Como tilizar o programa

1. Clone ou baixe os arquivos do repositório.
2. Abra o seu emlador de terminal de preferênciae navegue até a pasta raiz do projeto baixado.
3. Utilize o comando abaixo para restaurar as dependência do projeto.

    ```
    dotnet restore
    ```

4. Em seguida compile e execute o projeto com o comando:

    ```
    dotnet run --oject Calculadora.ConsoleApp
    ```
